# Task Manager App

## Tech Stack
- React
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features
- Register/Login
- Create Tasks
- Edit Tasks
- Delete Tasks
- Search Tasks
- Dashboard Statistics
